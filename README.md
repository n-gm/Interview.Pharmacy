# Аптека 
Реализовать контроллер(ы), который позволяет:
- Получить список актуальных типов товаров (Medicine) и их количество
- Получить один конкретный тип товара и его количество по идентификатору
- Создать новый тип товара
- Обновить данные по типу товара(имя, описание, количество)
- Обновить данные по типу товара(отдельно имя, отдельно описание, отдельно количество)
- Удалить определенный тип товара

Все решения по реализации сверх поставленной задачи на усмотрение разработчика;

При необходимости создать новые сервисы для какого-либо важного функционала можно ограничиться созданием интерфейса с описанием свойств/методов;

Необходимо реализовать максимум функционала за отведенное время. Словесно обозначить:
- Проблемы, которые есть в имеющемся коде с потенциальными решениями проблемы. В том числе если это не проблема, но можно улучшить;
- Что было бы важно реализовать, но это не реализовываем сейчас из-за недостатка времени (или иных причин) с аргументами почему этот функционал можно принести в жертву;
- Что было бы неплохо реализовать, но это не реализовываем сейчас из-за недостатка времени (или иных причин) с аргументами почему этот функционал можно пропустить;
